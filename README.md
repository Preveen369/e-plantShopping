# E-Plant Shopping
E-Plant Shopping is a React-based e-commerce web application designed for plant enthusiasts to discover, select, and purchase a variety of indoor and outdoor plants. It offers an intuitive interface for browsing plant catalogs, managing shopping carts, and simulating secure checkouts, with detailed product information including care tips and pricing.

## ğŸ§  How It Works
1. Users land on the home page and browse the plant catalog or use the search bar to find specific items.
2. Select a plant to view details, then add it to the cart using the "Add to Cart" button.
3. Access the cart to review items, adjust quantities, or proceed to checkout.
4. During checkout, enter shipping info and select payment method for a simulated order confirmation.
5. All cart data is managed locally for a smooth, offline-capable experience.
